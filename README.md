# Actividad9
Actividad Servidor simple con node.js
Proyecto Node.js - Servidor Básico
Este repositorio contiene un proyecto práctico desarrollado para la asignatura de programación web, donde se implementa un servidor web utilizando el módulo nativo http de Node.js.

🚀 Características
Servidor HTTP Nativo: Configuración de un servidor sin dependencias externas (Express).

Lógica Matemática: Implementación de funciones para operaciones de suma y multiplicación.

Estructuras de Control: Uso de if/else para validación de resultados y un ciclo for para generación dinámica de contenido.

Manejo de Archivos (fs): Registro automático de visitas en un archivo local log_visitas.txt.

🛠️ Tecnologías utilizadas
Node.js (Entorno de ejecución)

JavaScript (ES6+)

File System (fs) (Módulo nativo para persistencia de datos)

📋 Requisitos
Para ejecutar este proyecto, necesitas tener instalado:

Node.js (Versión 16 o superior recomendada)

🔧 Instalación y Ejecución
Clona el repositorio o descarga los archivos.

Abre una terminal en la carpeta del proyecto.

Ejecuta el servidor con el siguiente comando:
npm init -y
Instalar mysql2 -- npm install mysql2
instalar ejs html js -- npm install ejs

npm install express ejs mysql2 express-session

node app.js
Abre tu navegador y accede a: http://localhost:3000
